# 项目计划: JD-Insight (AI宠物版 - 黑客马拉松)

## 1. 目标与策略

**首要目标**: 在黑客马拉松期限内，构建一个功能完善且**极具创意**的JD-Insight AI宠物扩展。项目不仅要展示设备端AI的强大功能，更要通过有趣的游戏化体验给评委留下深刻印象。

**策略**:
1.  **功能优先，主题并行**: 核心AI分析流程作为主线任务。在此基础上，并行设计和开发宠物主题的UI和动画。
2.  **快速迭代**: 使用原生JS和CSS动画，避免复杂的库和框架，确保开发速度。
3.  **聚焦核心体验**: 重点打磨“喂食 -> 消化 -> 产出宝石 -> 宝石融合”这一核心互动循环。

---

## 3. 黑客马拉松成功的关键因素

*   **一个会讲故事的演示视频**: 视频是成败关键。一定要从“认识你的AI求职宠物”开始，完整地讲述一个“喂养->成长->回报”的有趣故事。
*   **动画的流畅性**: 宠物的互动动画是创意的核心，确保它们看起来精致、不卡顿。
*   **清晰的核心价值**: 在趣味性之外，清晰地展示最终“技能水晶”的价值——它如何为求职者提供了清晰的技能洞察。
